# API Employee DevOps

Este proyecto es una API RESTful para la gestión de empleados, construida con Java y Spring Boot. Incorpora prácticas modernas de DevOps incluyendo integración continua con Jenkins, análisis de código con SonarQube y contenedorización con Docker.

## Tecnologías Utilizadas

*   **Lenguaje**: Java 17
*   **Framework**: Spring Boot 3.x.x
*   **Base de Datos**: [H2 Database](https://www.h2database.com/) (En memoria)
*   **Documentación API**: [SpringDoc OpenAPI (Swagger)](https://springdoc.org/)
*   **Mapeo de Objetos**: [MapStruct](https://mapstruct.org/)
*   **Reducción de Código**: [Lombok](https://projectlombok.org/)
*   **Pruebas**: JUnit 5, Mockito
*   **DevOps**: Docker, GitHub Actions, SonarCloud

## Pre-requisitos

Asegúrate de tener instalado lo siguiente en tu entorno local:

*   **Java JDK 17** o superior.
*   **Maven 3.8.x** o superior.
*   **Docker** (Opcional, para ejecución en contenedores).
*   **Git**.
